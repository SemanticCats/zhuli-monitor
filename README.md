# 🚀 查主力监控系统  (全自动掘金版)

这是一个基于 GitHub Actions 的自动化股票监控系统。它利用全市场大数据扫描，结合**筹码峰分析**与**板块ETF交叉验证**，自动挖掘主力资金正在操盘的潜力股。

> **⏰ 推送时间**：每个交易日 18:00 (北京时间)
> **📤 推送方式**：微信服务通知 (通过 Server酱)

---

## 📋 核心功能

### 🔍 全市场自动挖掘
不再局限于“自选股”，系统会自动扫描全市场热点板块，从中捕捉主力资金介入的个股。

### 📊 筹码峰分析
通过分析换手率和成交量分布，判断主力是否已经“锁仓”（筹码集中）。只有主力锁仓的票，系统才会关注。

### 🛡️ ETF交叉验证 (关键升级)
**过滤假信号**：不仅看个股，还要看该板块的ETF（如AI板块看AIETF）。只有当**板块ETF强势**且**个股筹码集中**时，才会发出买入信号。

### 📅 每日复盘推送
每天下午6点自动运行，生成报告并推送到你的微信，指导第二天的操作策略。

---

## ⚙️ 运行原理

系统由三个核心文件组成：

1.  **`main.py`** (大脑)：
    *   负责获取全市场数据。
    *   执行“ETF验证 + 筹码峰分析”的核心算法。
2.  **`.github/workflows/run.yml`** (闹钟)：
    *   设置每天 18:00 (UTC 10:00) 自动唤醒服务器运行代码。
3.  **`requirements.txt`** (工具箱)：
    *   列出了运行所需的第三方库（如 akshare, pandas）。

## 📱 部署步骤 (回顾)

如果你需要重新部署，请参考以下步骤：

1.  **创建仓库**：在 GitHub 创建一个新的 Public 仓库。
2.  **上传文件**：依次创建并上传以下文件：
    *   `requirements.txt`
    *   `main.py`
    *   `.github/workflows/run.yml`
    *   `README.md` (就是本文件)
3.  **配置密钥**：
    *   进入仓库 `Settings` -> `Secrets and variables` -> `Actions`。
    *   点击 `New repository secret`。
    *   **Name**: `SERVER_SENDKEY`
    *   **Value**: 你的 SCT密钥
4.  **坐等推送**：每天下午 6 点查看微信即可。
